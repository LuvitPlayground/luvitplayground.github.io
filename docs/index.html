Table of Contents

There are several types of documentation available on this website:

  • An introductory tutorial to help new users get started developing Luvit applications (when in doubt, start here)
  • Guides written by the community can show you how to solve problems that are commonly encountered
  • Explanations of important concepts, intended for those who want to dive deeper and understand how things work in detail
  • References for all of the Luvit core components and the functionality that they make available via APIs or other means
  • A list of libraries that are incorporated in the Luvit core can be found below, in case you want to learn more about their use

If what you're looking for isn't included here, please open an issue or let us know in the Discord server!

Dependencies#

  • LuaJIT is the state-of-the-Art Lua engine that forms the basis of the Luvi runtime
  • libuv controls the event loop, and provides Luvi-based applications with asynchronous I/O facilities
  • OpenSSL implements the underlying functionality for cryptography and hashing
  • miniz is included for the exported compression and decompression algorithms
